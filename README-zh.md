# MineCraft Signal
让Minecraft处理操作系统信号
# 这是什么
当你发送一个"信号"给Minecraft服务器进程时，它会运行你定义好的指令
例如，你可以定义SIGUSR1信号为/stop指令，当MC服务器接收到这个信号时，他将运行这个指令来安全地停止服务器
